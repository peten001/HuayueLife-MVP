# P10 USB 打印机门店验证清单

> “P10”仅沿用用户当前称呼；品牌和型号必须以 Android APP 读取的 `Build.MANUFACTURER`、`Build.MODEL` 为准。本清单不预设打印机型号、VID、PID、interface、endpoint、纸宽或切刀能力。

本轮仅验证：Android Web 收银外壳可启动、USB Host 能识别设备、用户授权后可以安全发送本地 ESC/POS 测试小票。

不得使用真实订单或顾客信息，不得测试开钱箱，不得连接云桥生产打印任务，不得开启自动打印。

## F. 前台稳定性

本阶段不验证自动打印、后台打印或开机恢复。以下每次打印必须由测试人员单独点击。

| 测试项 | 预期 | 实际 | 结果 | 错误码 | 截图/照片 | 备注 |
|---|---|---|---|---|---|---|
| 人工连续3次 | 每次等待完成后再点击，共3张且不多不少 |  |  |  |  |  |
| 打印中快速连点 | 不产生第二个并发写入 |  |  |  |  |  |
| 打印中拔出 | 停止并显示 `USB_DEVICE_DETACHED`，APP不崩溃 |  |  |  |  |  |
| 拔出再插入 | 刷新、重新授权后可再次手动测试 |  |  |  |  |  |
| 切换后台 | 当前诊断动作停止，不在后台继续或自动恢复打印 |  |  |  |  |  |
| 返回前台 | 可刷新设备并重新手动测试 |  |  |  |  |  |
