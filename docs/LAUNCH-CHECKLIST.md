# V1.0 Alpha 上线检查表

- [ ] `manifest.json` 已填写真实微信 AppID。
- [ ] `pages.json` 包含全部业务页面。
- [ ] TabBar 为：首页 / 订单 / 我的。
- [ ] 定位权限和隐私保护指引已配置。

- [ ] `NODE_ENV=production`。
- [ ] `JWT_SECRET` 已更换为至少 32 字节的随机值。
- [ ] `WECHAT_APP_ID` 和 `WECHAT_APP_SECRET` 已安全配置。
- [ ] `MINIAPP_QR_ENTRY_URL` 使用正式 HTTPS 地址。
