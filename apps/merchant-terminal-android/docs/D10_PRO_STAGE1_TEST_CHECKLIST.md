| DNS 失败 | 错误页显示 DNS 类错误和简要错误码 |  |  |  |  |

| HTTP 主文档错误 | 受控测试环境返回 4xx/5xx 时，显示 HTTP 错误及状态码 |  |  |  |  |
| SSL 错误 | 受控测试环境使用无效证书时连接被取消；APP 不绕过证书校验 |  |  |  |  |
| 错误页重新加载 | 点击重新加载只触发一次加载，成功后错误页消失 |  |  |  |  |
| 打开网络设置 | 错误页或诊断页入口可打开系统网络设置，返回 APP 正常 |  |  |  |  |
| WebView 渲染进程异常 | 受控调试触发后显示渲染进程错误，可重新创建/加载页面且 APP 不崩溃 |  |  |  |  |

> DNS、SSL、HTTP 和渲染进程异常测试必须在受控 debug 环境完成，不要攻击生产服务，也不要为了测试修改生产 DNS 或忽略 SSL 错误。
